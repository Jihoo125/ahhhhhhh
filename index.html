API Gateway URL을 설정하면 이 폼을 바로 AWS로 연결할 수 있습니다.

AWS 연결 방법

정적 웹사이트를 실제 AWS 백엔드와 연결하는 권장 구성

이 페이지는 S3 + CloudFront에 배포하고, 폼 제출은 API Gateway와 Lambda로 보내며, 인증은 Cognito를 통해 붙이는 방식으로 확장할 수 있습니다.

1. 정적 배포

`index.html`, `styles.css`, `script.js`를 S3 버킷에 업로드하고 CloudFront를 앞단에 둡니다.

2. API 연결

API Gateway 엔드포인트를 `window.NOTEFLOW_CONFIG.apiBaseUrl`로 주입하면 폼이 Lambda로 POST 됩니다.

3. 인증 연결

Cognito User Pool 정보를 같은 설정 객체에 넣어 로그인 버튼과 보호 API 호출로 확장할 수 있습니다.
